# J.A.R.V.I.S
🤖 J.A.R.V.I.S is a futuristic AI desktop assistant built with Python and PyQt6, powered by Groq LLaMA 3.3 70B. It enables voice control, system automation, file analysis, and real-time monitoring through a sci-fi HUD interface, turning your Linux desktop into an intelligent, interactive control system.
